# Code of Conduct

BitRep is an open protocol and community. To keep collaboration productive and respectful, all contributors are expected to follow this Code of Conduct.

## Standards
Participants are expected to:
- communicate clearly and constructively
- respect differing viewpoints
- focus on technical issues, not individuals
- provide helpful feedback without hostility
- assume good faith in discussions

## Unacceptable Behavior
The following are not permitted:
- personal attacks or harassment
- discriminatory language or conduct
- sustained disruption of discussions
- publishing private information without consent
- aggressive or bad‑faith argumentation

## Scope
This Code of Conduct applies to:
- the repository and issue tracker
- pull requests and code reviews
- project discussions and communication channels

## Enforcement
Violations may result in:
- a warning
- temporary interaction limits
- removal from discussions
- in severe cases, a ban from participation

## Reporting
If you experience or witness unacceptable behavior, report it privately to the maintainer:

**conduct@bitrep.dev**  
(If not active, use the maintainer’s GitHub‑associated email.)

Reports will be reviewed promptly and confidentially.

## Commitment
By participating in BitRep, you agree to uphold this Code of Conduct and contribute to a respectful, focused, and collaborative environment.
